# ✊✋✌ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML**, **CSS**, and **JavaScript**. The game randomly generates moves for both players, determines the winner based on the game rules, and displays the result with a clean user interface.

---

## 🚀 Features

- 🎮 One-click gameplay
- 🎲 Random move generation
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Reset button to play again
- 🎨 Dynamic UI updates
- 📱 Simple and responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 🎯 Game Rules

- 🪨 Rock beats Scissors
- 📄 Paper beats Rock
- ✂️ Scissors beat Paper
- If both players choose the same option, the game ends in a Draw.

---

## ▶️ How to Play

1. Click the **Play** button.
2. Both players randomly receive one of:
   - Rock
   - Paper
   - Scissors
3. The winner is displayed based on the game rules.
4. Click **Reset** to start a new round.
